\doxysubsection{Detailed Description}
Implementation of the \doxylink{class_black_scholes_pricer}{Black\+Scholes\+Pricer} class using the Black-\/\+Scholes formula. 

This file implements the \doxylink{class_black_scholes_pricer}{Black\+Scholes\+Pricer} class which derives from the \doxylink{class_i_option_pricer}{IOption\+Pricer} interface. It calculates the price and \doxylink{struct_greeks}{Greeks} (delta, gamma, vega, theta, and rho) of a European option using the Black-\/\+Scholes formula with continuous dividends.
