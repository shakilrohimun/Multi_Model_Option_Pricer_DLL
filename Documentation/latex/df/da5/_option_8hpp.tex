\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_option}{Option} class and the \doxylink{struct_greeks}{Greeks} structure for option pricing. 

This file contains the declaration of the \doxylink{class_option}{Option} class, which represents a financial derivative, and the \doxylink{struct_greeks}{Greeks} structure, which stores various sensitivity measures (\doxylink{struct_greeks}{Greeks}) used in option pricing. These measures help in assessing how the option\textquotesingle{}s value changes in response to variations in underlying parameters.
