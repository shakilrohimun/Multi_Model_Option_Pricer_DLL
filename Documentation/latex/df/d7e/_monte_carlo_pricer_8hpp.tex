\doxysubsection{Detailed Description}
Declaration and detailed documentation of the \doxylink{class_monte_carlo_pricer}{Monte\+Carlo\+Pricer} class. 

This header file declares the \doxylink{class_monte_carlo_pricer}{Monte\+Carlo\+Pricer} class, which implements option pricing using Monte Carlo simulation techniques. The class simulates numerous potential future price paths of the underlying asset to compute the expected payoff of the option.

For European options, a standard Monte Carlo simulation is used. For American options, the Longstaff-\/\+Schwartz method (which employs least-\/squares regression) is utilized to determine the optimal early exercise strategy.

This approach is particularly useful when closed-\/form solutions are not available, providing a flexible and robust method for pricing a variety of option types.
