\doxysubsection{Detailed Description}
Class for managing the interest rate curve. 

The \doxylink{class_yield_curve}{Yield\+Curve} class manages a collection of \doxylink{struct_rate_point}{Rate\+Point} objects, which together form an interest rate curve. It provides methods to add new rate points, interpolate the interest rate for a specified maturity, access the underlying data, and load rate data from a file. This is essential for financial models that require variable interest rates, such as when discounting cash flows or pricing interest rate-\/sensitive derivatives.

The interpolation is typically performed using linear interpolation between the nearest known rate points. The load\+From\+File function expects a text file where each line contains two numbers separated by whitespace\+: the maturity (as a fraction between 0 and 1) and the corresponding interest rate.
