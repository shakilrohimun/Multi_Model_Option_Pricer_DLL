\doxysubsection{Detailed Description}
Factory for creating pricing engine instances. 

The \doxylink{class_pricer_factory}{Pricer\+Factory} class provides static methods to create instances of pricing engines (implementations of \doxylink{class_i_option_pricer}{IOption\+Pricer}) based on the specified pricer type and configuration. This design pattern promotes flexibility and scalability in the option pricing system.
