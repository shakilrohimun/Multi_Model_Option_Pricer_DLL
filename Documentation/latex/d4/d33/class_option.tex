\doxysubsection{Detailed Description}
Class representing a financial option. 

The \doxylink{class_option}{Option} class encapsulates the essential parameters needed for option pricing, including the underlying asset\textquotesingle{}s price, strike price, volatility, dividend yield (if any), option type (Call or Put), and option style (European or American).

It provides methods to access (getters) and modify (setters) these parameters, enabling flexibility in option pricing calculations.
