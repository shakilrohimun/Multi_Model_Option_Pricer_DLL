\doxysubsection{Detailed Description}
Class implementing the Monte Carlo pricing model. 

The \doxylink{class_monte_carlo_pricer}{Monte\+Carlo\+Pricer} class derives from the \doxylink{class_i_option_pricer}{IOption\+Pricer} interface and provides concrete implementations for calculating both the option price and its \doxylink{struct_greeks}{Greeks} using Monte Carlo simulation.

The class supports two primary simulation methods\+:
\begin{DoxyItemize}
\item A standard Monte Carlo simulation for European options.
\item The Longstaff-\/\+Schwartz method for American options, which uses least-\/squares regression to determine the optimal early exercise strategy.
\end{DoxyItemize}

The simulation parameters such as the number of simulation paths and the time steps per path are configurable via the \doxylink{struct_pricing_configuration}{Pricing\+Configuration} structure, allowing the user to balance precision and performance.

Computes the price of the option using Monte Carlo simulation. 

Computes the option price using Monte Carlo simulation.

This method simulates multiple future price paths for the underlying asset and calculates the expected payoff of the option by discounting and averaging the results from all simulated paths.


\begin{DoxyParams}{Parameters}
{\em opt} & A constant reference to an \doxylink{class_option}{Option} object containing parameters such as the underlying asset price, strike price, volatility, dividend yield, and maturity. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The computed option price as a double.
\end{DoxyReturn}
For American options, the Longstaff-\/\+Schwartz algorithm is employed to handle early exercise, while for European options, the option payoff is directly calculated at maturity.

For European options, a standard simulation of geometric Brownian motion is used. For American options, the Longstaff-\/\+Schwartz algorithm is applied to determine the optimal exercise.

The simulation parameters (maturity, risk-\/free rate, number of simulation paths, and number of time steps) are obtained from the \doxylink{struct_pricing_configuration}{Pricing\+Configuration} object. If a calculation date is provided, the effective time to maturity is adjusted as\+:

T\+\_\+effective = T -\/ offset,

where offset is the number of years between config\+\_\+.\+calculation\+Date and today.

In the forward simulation, the local risk-\/free rate at each step is obtained by\+: r\+\_\+local = (if yield\+Curve is available) ? yield\+Curve.\+get\+Rate(t\+\_\+norm) \+: default risk\+Free\+Rate, where t\+\_\+norm is the normalized time (current time / T\+\_\+effective).

This member variable holds a \doxylink{struct_pricing_configuration}{Pricing\+Configuration} object that contains various simulation parameters, including calculation date, risk-\/free rate, number of simulation paths, and time steps per path.
