\doxysubsection{Detailed Description}
A utility class for performing date conversions and calculations. 

This class provides static methods to convert between date strings and std\+::chrono time points, as well as to compute time differences expressed in years. The functions in this class are designed to handle dates in the ISO 8601 format (\"{}\+YYYY-\/\+MM-\/\+DD\"{}) and leverage the C++ standard library\textquotesingle{}s $<$chrono$>$ facilities to represent and manipulate time.

\begin{DoxyNote}{Note}
All methods in this class are static and can be used without instantiating an object.
\end{DoxyNote}
