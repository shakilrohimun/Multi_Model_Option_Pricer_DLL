\doxysubsection{Detailed Description}
Interface for an option pricing engine. 

The \doxylink{class_i_option_pricer}{IOption\+Pricer} interface specifies the necessary methods for pricing options and computing their associated \doxylink{struct_greeks}{Greeks}. Any pricing engine that derives from this interface must implement these methods to allow for consistent integration within the pricing system.

\begin{DoxyNote}{Note}
The interface uses pure virtual functions to enforce implementation in derived classes. 
\end{DoxyNote}
