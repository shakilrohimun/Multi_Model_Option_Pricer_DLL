Computes the option price using the Crank-\/\+Nicolson finite difference method. 

Computes the \doxylink{struct_greeks}{Greeks} of an option using the Crank-\/\+Nicolson method.

This function calculates the theoretical price of an option by numerically solving the Black-\/\+Scholes partial differential equation using the Crank-\/\+Nicolson scheme. It discretizes both the time to maturity and the underlying asset price into a finite grid and applies an implicit method that offers a balance between stability and accuracy.

The Crank-\/\+Nicolson method is an implicit finite difference technique that averages the results of the explicit and implicit schemes. This method is particularly effective for option pricing as it provides stability even with larger time steps, making it suitable for a wide range of applications in financial modeling.

This function calculates the sensitivity measures (\doxylink{struct_greeks}{Greeks}) of an option\textquotesingle{}s price with respect to changes in various parameters by employing the Crank-\/\+Nicolson finite difference method. The computed \doxylink{struct_greeks}{Greeks} include\+: Delta, Gamma, Vega, Theta, and Rho.

The function applies finite difference approximations on the grid produced by the Crank-\/\+Nicolson method to evaluate the partial derivatives of the option price with respect to the input parameters. These \doxylink{struct_greeks}{Greeks} are essential for risk management, hedging strategies, and understanding how the option\textquotesingle{}s price is affected by market conditions.
