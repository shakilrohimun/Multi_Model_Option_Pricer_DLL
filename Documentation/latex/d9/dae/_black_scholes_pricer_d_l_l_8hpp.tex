Calculates the price of an option using the Black-\/\+Scholes formula. 

Computes the \doxylink{struct_greeks}{Greeks} of an option using the Black-\/\+Scholes model.

This function computes the theoretical price of a European option based on the Black-\/\+Scholes model. It evaluates the necessary parameters using cumulative distribution functions for the standard normal distribution.

The Black-\/\+Scholes model is based on several assumptions including constant volatility and interest rates, as well as a log-\/normal distribution of returns. It is widely used for pricing European options where early exercise is not permitted.

This function calculates the sensitivity measures (\doxylink{struct_greeks}{Greeks}) that describe how the price of an option responds to changes in various underlying parameters. The \doxylink{struct_greeks}{Greeks} provided include\+:
\begin{DoxyItemize}
\item Delta\+: Sensitivity of the option price to changes in the underlying asset\textquotesingle{}s price.
\item Gamma\+: Sensitivity of Delta with respect to changes in the underlying asset\textquotesingle{}s price.
\item Vega\+: Sensitivity of the option price to changes in volatility.
\item Theta\+: Sensitivity of the option price to the passage of time.
\item Rho\+: Sensitivity of the option price to changes in the risk-\/free interest rate.
\end{DoxyItemize}
