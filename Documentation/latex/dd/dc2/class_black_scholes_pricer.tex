\doxysubsection{Detailed Description}
A class that implements the Black-\/\+Scholes pricing model for European options. 

The \doxylink{class_black_scholes_pricer}{Black\+Scholes\+Pricer} class is derived from the \doxylink{class_i_option_pricer}{IOption\+Pricer} interface, ensuring that it provides the standard methods for option pricing and computing \doxylink{struct_greeks}{Greeks}. This class encapsulates the Black-\/\+Scholes formula logic and integrates market configuration parameters via the \doxylink{struct_pricing_configuration}{Pricing\+Configuration} object.

The Black-\/\+Scholes model assumes that the underlying asset follows a geometric Brownian motion with constant drift and volatility. This analytical model is used to compute the theoretical price of European options and their sensitivity to various market factors, thus playing a crucial role in risk management and hedging.

Computes the \doxylink{struct_greeks}{Greeks} for a European option using the Black-\/\+Scholes model. 

Computes the \doxylink{struct_greeks}{Greeks} of the option using the Black-\/\+Scholes model.

This method calculates various sensitivity measures (\doxylink{struct_greeks}{Greeks}) of the option\textquotesingle{}s price to changes in underlying market parameters. The \doxylink{struct_greeks}{Greeks} computed include\+:
\begin{DoxyItemize}
\item Delta\+: Rate of change of the option price with respect to changes in the underlying asset\textquotesingle{}s price.
\item Gamma\+: Rate of change of Delta with respect to changes in the underlying asset\textquotesingle{}s price.
\item Vega\+: Sensitivity of the option price to changes in volatility.
\item Theta\+: Rate of decline of the option price with respect to time.
\item Rho\+: Sensitivity of the option price to changes in the risk-\/free interest rate.
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}

\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A \doxylink{struct_greeks}{Greeks} structure that encapsulates the computed values of Delta, Gamma, Vega, Theta, and Rho. 
\end{DoxyReturn}

\begin{DoxyExceptions}{Exceptions}
{\em std\+::runtime\+\_\+error} & if the option is not European, since the Black-\/\+Scholes model is not designed for American options.\\
\hline
\end{DoxyExceptions}
The method leverages the classical analytical formulas for each Greek, which involve differentiating the Black-\/\+Scholes pricing equation with respect to the underlying parameters. This provides critical insights into how the option price reacts to market movements, supporting effective risk management and hedging strategies.

This method calculates the following \doxylink{struct_greeks}{Greeks} for a European option\+:
\begin{DoxyItemize}
\item Delta\+: Sensitivity of the price with respect to the underlying.
\item Gamma\+: Sensitivity of Delta with respect to the underlying.
\item Vega\+: Sensitivity with respect to volatility.
\item Theta\+: Sensitivity with respect to time.
\item Rho\+: Sensitivity with respect to the risk-\/free rate.
\end{DoxyItemize}

Maturity and risk-\/free rate are taken from the configuration, and the effective time to maturity is adjusted by the calculation (continuation) date if provided. The effective risk-\/free rate is computed by averaging the yield curve data (if available) or using the default value.

Computes the price of a European option using the Black-\/\+Scholes formula. 

Computes the option price using the Black-\/\+Scholes formula.

This method calculates the theoretical price of a European option based on the Black-\/\+Scholes model. It utilizes both the option parameters (e.\+g., underlying price, strike price, volatility) and the configuration parameters (e.\+g., risk-\/free rate, maturity) provided in the \doxylink{struct_pricing_configuration}{Pricing\+Configuration}.

The pricing method involves computing intermediate variables (such as d1 and d2) that are central to the Black-\/\+Scholes formula. These variables are then used to evaluate the cumulative distribution functions of the standard normal distribution, yielding the option price.

This method calculates the price of a European option using the Black-\/\+Scholes formula. It retrieves the maturity and risk-\/free rate from the configuration and adjusts the effective time to maturity based on the calculation (continuation) date if provided. It then computes an effective risk-\/free rate by averaging the rates from the yield curve (if available). If no yield curve data is loaded, the default risk-\/free rate is used.
