\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_yield_curve}{Yield\+Curve} class and \doxylink{struct_rate_point}{Rate\+Point} structure for managing the interest rate curve. 

This file defines the \doxylink{struct_rate_point}{Rate\+Point} structure and the \doxylink{class_yield_curve}{Yield\+Curve} class, which provide a mechanism to store and manage interest rate data as a function of time (maturity). The \doxylink{class_yield_curve}{Yield\+Curve} class offers functionalities such as adding rate points, interpolating interest rates for a given maturity, retrieving stored data, and loading rate data from an external file.
