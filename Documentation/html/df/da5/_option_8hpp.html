Detailed Description

Declaration of the Option class and the Greeks structure for option pricing.

This file contains the declaration of the Option class, which represents a financial derivative, and the Greeks structure, which stores various sensitivity measures (Greeks) used in option pricing. These measures help in assessing how the option's value changes in response to variations in underlying parameters.
