Detailed Description

Implementation de la classe Option et de la structure Greeks pour le pricing d'options.

Ce fichier contient l'implementation complete de la classe Option, utilisee pour representer une option financiere avec ses parametres de base (sous-jacent, strike, volatilite, dividende, type et style). Les methodes getter et setter permettent de manipuler ces parametres.
