Detailed Description

Class representing a financial option.

The Option class encapsulates the essential parameters needed for option pricing, including the underlying asset's price, strike price, volatility, dividend yield (if any), option type (Call or Put), and option style (European or American).

It provides methods to access (getters) and modify (setters) these parameters, enabling flexibility in option pricing calculations.

Specifies whether an option is European or American:

  • European: Can only be exercised at maturity.
  • American: Can be exercised at any time up to maturity.
