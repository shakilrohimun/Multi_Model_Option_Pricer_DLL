Detailed Description

A class that implements the Black-Scholes pricing model for European options.

The BlackScholesPricer class is derived from the IOptionPricer interface, ensuring that it provides the standard methods for option pricing and computing Greeks. This class encapsulates the Black-Scholes formula logic and integrates market configuration parameters via the PricingConfiguration object.

The Black-Scholes model assumes that the underlying asset follows a geometric Brownian motion with constant drift and volatility. This analytical model is used to compute the theoretical price of European options and their sensitivity to various market factors, thus playing a crucial role in risk management and hedging.

Greeks BlackScholesPricer::computeGreeks ( const Option &  opt) const

Computes the Greeks for a European option using the Black-Scholes model.

Computes the Greeks of the option using the Black-Scholes model.

This method calculates various sensitivity measures (Greeks) of the option's price to changes in underlying market parameters. The Greeks computed include:

  • Delta: Rate of change of the option price with respect to changes in the underlying asset's price.
  • Gamma: Rate of change of Delta with respect to changes in the underlying asset's price.
  • Vega: Sensitivity of the option price to changes in volatility.
  • Theta: Rate of decline of the option price with respect to time.
  • Rho: Sensitivity of the option price to changes in the risk-free interest rate.
Parameters
optA constant reference to an Option object containing the details required for pricing.
Returns
A Greeks structure that encapsulates the computed values of Delta, Gamma, Vega, Theta, and Rho.
Exceptions
std::runtime_errorif the option is not European, since the Black-Scholes model is not designed for American options.

The method leverages the classical analytical formulas for each Greek, which involve differentiating the Black-Scholes pricing equation with respect to the underlying parameters. This provides critical insights into how the option price reacts to market movements, supporting effective risk management and hedging strategies.

This method calculates the following Greeks for a European option:

  • Delta: Sensitivity of the price with respect to the underlying.
  • Gamma: Sensitivity of Delta with respect to the underlying.
  • Vega: Sensitivity with respect to volatility.
  • Theta: Sensitivity with respect to time.
  • Rho: Sensitivity with respect to the risk-free rate.

Maturity and risk-free rate are taken from the configuration, and the effective time to maturity is adjusted by the calculation (continuation) date if provided. The effective risk-free rate is computed by averaging the yield curve data (if available) or using the default value.

Parameters
optThe option to evaluate.
Returns
A Greeks structure containing the calculated values.
Exceptions
std::runtime_errorif the option is not European.

Computes the price of a European option using the Black-Scholes formula.

Computes the option price using the Black-Scholes formula.

This method calculates the theoretical price of a European option based on the Black-Scholes model. It utilizes both the option parameters (e.g., underlying price, strike price, volatility) and the configuration parameters (e.g., risk-free rate, maturity) provided in the PricingConfiguration.

Parameters
optA constant reference to an Option object that encapsulates the essential option parameters.
Returns
A double representing the calculated option price.
Exceptions
std::runtime_errorif the provided option is not European, as the Black-Scholes model is applicable only to European style options.

The pricing method involves computing intermediate variables (such as d1 and d2) that are central to the Black-Scholes formula. These variables are then used to evaluate the cumulative distribution functions of the standard normal distribution, yielding the option price.

This method calculates the price of a European option using the Black-Scholes formula. It retrieves the maturity and risk-free rate from the configuration and adjusts the effective time to maturity based on the calculation (continuation) date if provided. It then computes an effective risk-free rate by averaging the rates from the yield curve (if available). If no yield curve data is loaded, the default risk-free rate is used.

The computed option price.
