Computes the price of an option using the Monte Carlo simulation model.

Computes the Greeks of an option using the Monte Carlo simulation model.

This function calculates the theoretical price of an option by generating multiple simulated price paths for the underlying asset. For European options, the payoff is evaluated at maturity, while for American options, simulation techniques that allow for early exercise decisions are employed.

Parameters
SThe current price of the underlying asset.
KThe strike price of the option.
TThe time to maturity (in years).
rThe risk-free interest rate.
sigmaThe volatility of the underlying asset.
qThe dividend yield of the underlying asset.
optionTypeAn integer indicating the type of option (0 for Call, 1 for Put).
optionStyleAn integer indicating the style of the option (0 for European, 1 for American).
calculationDateA null-terminated string representing the calculation date in "YYYY-MM-DD" format. If empty, the current date is used.
mcNumPathsThe number of simulation paths to generate.
mcTimeStepsPerPathThe number of time steps for each simulation path.
Returns
The computed option price as a double.

This function calculates key sensitivity measures (Greeks) such as Delta, Gamma, Vega, Theta, and Rho by simulating numerous price paths for the underlying asset. The computed values are output via the pointer parameters provided.

Parameters
SThe current price of the underlying asset.
KThe strike price of the option.
TThe time to maturity (in years).
rThe risk-free interest rate.
sigmaThe volatility of the underlying asset.
qThe dividend yield of the underlying asset.
optionTypeAn integer representing the type of option (0 for Call, 1 for Put).
optionStyleAn integer representing the option style (0 for European, 1 for American).
calculationDateA null-terminated string in "YYYY-MM-DD" format representing the calculation date. If empty, the current date is used.
mcNumPathsThe number of simulation paths to generate.
mcTimeStepsPerPathThe number of time steps per simulation path.
deltaPointer to a double where the computed Delta will be stored.
