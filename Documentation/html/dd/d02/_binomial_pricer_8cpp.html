Detailed Description

Implementation of the BinomialPricer class using the CRR method.

This file implements the option pricing model using a binomial tree. It supports pricing for both European and American options. In addition to the basic option parameters, this pricer uses additional configuration parameters (e.g., calculation date, number of steps, maturity, risk-free rate, and yield curve) from a PricingConfiguration object.
