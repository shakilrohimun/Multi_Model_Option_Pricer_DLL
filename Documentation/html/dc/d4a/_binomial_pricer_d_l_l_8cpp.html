Computes the Greeks of an option using the binomial model.

This function sets up the pricing configuration, loads the yield curve data, creates a BinomialPricer instance, and computes the Greeks (delta, gamma, vega, theta, rho) by applying finite differences.

Parameters
SThe underlying asset price.
KThe strike price.
TThe maturity (in years).
rThe risk-free interest rate.
sigmaThe volatility of the underlying asset.
qThe dividend yield.
optionTypeAn integer representing the option type (0 for Call, 1 for Put).
optionStyleAn integer representing the option style (0 for European, 1 for American).
calculationDateA C-string representing the calculation date in "YYYY-MM-DD" format. If empty, today's date is used.
binomialStepsThe number of steps in the binomial tree.
deltaPointer to store the computed delta.
gammaPointer to store the computed gamma.
vegaPointer to store the computed vega.
thetaPointer to store the computed theta.
rhoPointer to store the computed rho.

Computes the price of an option using the binomial model.

This function sets up the pricing configuration, loads the yield curve data, creates a BinomialPricer instance, and then computes the option price using the provided parameters.

Parameters
SThe underlying asset price.
KThe strike price.
TThe maturity (in years).
rThe risk-free interest rate.
sigmaThe volatility of the underlying asset.
qThe dividend yield.
optionTypeAn integer representing the option type (0 for Call, 1 for Put).
optionStyleAn integer representing the option style (0 for European, 1 for American).
calculationDateA C-string representing the calculation date in "YYYY-MM-DD" format. If empty, today's date is used.
binomialStepsThe number of steps in the binomial tree.
Returns
The computed option price, or -1.0 if an error occurs.
