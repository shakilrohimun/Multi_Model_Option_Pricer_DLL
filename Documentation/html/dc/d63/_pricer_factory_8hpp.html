Detailed Description

Declaration of the factory for creating pricing engines.

This header defines the PricerFactory class, which provides static methods to dynamically instantiate different types of option pricing engines based on the specified pricer type. The factory encapsulates the creation logic, thereby promoting a clean separation of concerns and ease of extension.

Enumeration of pricer types.

The PricerType enumeration defines the various pricing engine implementations that are supported. Each enumerated value corresponds to a specific pricing model used to compute option prices and Greeks.
