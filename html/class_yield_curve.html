Detailed Description

Class for managing the interest rate curve.

This class stores a series of points (maturity, rate) and provides methods for interpolating the interest rate for a given time. It also allows loading the data from a text file. Each line in the file should contain two numbers: the maturity (between 0 and 1) and the corresponding interest rate.
