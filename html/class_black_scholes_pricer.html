Greeks BlackScholesPricer::computeGreeks ( const Option &  opt) const
overridevirtual

Computes the Greeks of the option using the Black-Scholes model.

This method computes the following Greeks:

  • Delta: sensitivity with respect to the underlying price.
  • Gamma: sensitivity of Delta with respect to the underlying price.
  • Vega: sensitivity with respect to volatility.
  • Theta: sensitivity with respect to time.
  • Rho: sensitivity with respect to the risk-free rate.

The classical Black-Scholes formulas are used, with maturity and risk-free rate taken from the configuration.

Parameters
optThe option to evaluate.
Returns
A Greeks structure containing the calculated values.
Exceptions
std::runtime_errorif the option is not European.

This method calculates the following Greeks for a European option:

  • Delta: Sensitivity of the price with respect to the underlying.
  • Gamma: Sensitivity of Delta with respect to the underlying.
  • Vega: Sensitivity with respect to volatility.
  • Theta: Sensitivity with respect to time.
  • Rho: Sensitivity with respect to the risk-free rate.

Maturity and risk-free rate are taken from the configuration, and the effective time to maturity is adjusted by the calculation (continuation) date if provided. The effective risk-free rate is computed by averaging the yield curve data (if available) or using the default value.

Computes the option price using the Black-Scholes formula.

This method calculates the price of a European option using the Black-Scholes formula. The calculation uses the option parameters and the configuration parameters (maturity and risk-free rate).

This method calculates the price of a European option using the Black-Scholes formula. It retrieves the maturity and risk-free rate from the configuration and adjusts the effective time to maturity based on the calculation (continuation) date if provided. It then computes an effective risk-free rate by averaging the rates from the yield curve (if available). If no yield curve data is loaded, the default risk-free rate is used.
