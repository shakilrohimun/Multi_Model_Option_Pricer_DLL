double BinomialPricer::price ( const Option &  opt) const
overridevirtual

Computes the option price using the binomial CRR model.

This function calculates the price of an option using the binomial tree approach, with the number of steps specified in the pricing configuration.

Parameters
optThe option to be priced.
Returns
The computed option price.

This function calculates the price of an option using a binomial tree. The number of steps in the tree is taken from the configuration (config_.binomialSteps).

Instead of using a constant risk-free rate, the backward induction uses a variable rate obtained via interpolation from the yield curve stored in the configuration. For each time step, the local rate is obtained by: r_local = config_.yieldCurve.getRate(t_norm) where t_norm is the normalized time (between 0 and 1).

optThe option to be priced.
Returns
The computed option price.
