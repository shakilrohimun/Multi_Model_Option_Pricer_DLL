Implementation of the BlackScholesPricer class using the Black-Scholes formula.

This file implements the BlackScholesPricer class which derives from the IOptionPricer interface. It calculates the price and Greeks (delta, gamma, vega, theta, and rho) of a European option using the Black-Scholes formula with continuous dividends.
