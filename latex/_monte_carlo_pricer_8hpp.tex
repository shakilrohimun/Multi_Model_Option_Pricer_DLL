Declaration of the \doxylink{class_monte_carlo_pricer}{Monte\+Carlo\+Pricer} class. 

This class implements option pricing using Monte Carlo simulation. For European options, a standard Monte Carlo simulation is used. For American options, the Longstaff-\/\+Schwartz method (least-\/squares regression) is employed to determine optimal early exercise.
