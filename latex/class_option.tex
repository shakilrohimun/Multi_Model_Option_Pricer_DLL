\doxysubsection{Detailed Description}
Classe représentant une option financière. 

Cette classe encapsule les paramètres essentiels d\textquotesingle{}une option, tels que le prix du sous-\/jacent, le strike, la volatilité, le dividende éventuel, le type (Call/\+Put) et le style (Européenne/\+Americaine).
